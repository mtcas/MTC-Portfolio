Propert-Ease

Propert-Ease is a centralised platform that integrates all construction operations into one central hub, Propert-Ease has both web app (C# in Visual Studio) and mobile app (Kotlin in Android Studio) and works hand in hand with each other to provide users with similar design and exact concept to reduce confusion when switching from mobile app to web app. This team project monitors progress across multiple sites in real-time, automates reports and maintenance issues, allows users to assign/monitor/evaluate tasks, and track every operation the construction company does. Users are able to login/create accounts and select their roles, create projects that meet timelines and budgets, log issues with image/documentation, automate task assignment and performance tracking, upload secure documentation for the business, real-time notifications, and visualise operations.

Tech: C#, Kotlin, Visual Studio, Android Studio, SQL Servers, User Auth, .Net Framework, etc

Retail App (Microsoft Azure)

This project is a cloud-based order management web application built with ASP.NET MVC and hosted on Microsoft Azure. The application manages customers, products, and orders, storing data in Azure SQL Database while large files are stored in Azure Blob Storage. The web application runs on Azure App Service, with Azure Functions performing background tasks. Communication between services can use Azure Service Bus and Event Hubs, while Azure Traffic Manager, database replication, and backups ensure reliability, scalability, and high availability.
